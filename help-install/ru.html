Справка загрузчика

Welcome to @product@

Используйте это меню для выбора необходимой функции. Если у вас проблемы с управлением в данной системе помощи, нажмитеF1 чтобы войти в описание системы помощи. Главные функции в этом меню:

Загрузка с жёсткого диска: Этот выбор ничего ничего не сделает с системой. Просто будет запущена ранне установленная система.

Установка: Этот режим установки работает на большинстве компьютеров. Если у вас зависает система при загрузке или у вас проблемы с обнаружением аппаратных средств, таких как контроллеры диска или сетевые платы, попробуйте одну из следующих установочных опций.

Установка - ACPI выключен: Многие проданные компьюетры имеют неполную или неверную реализацию ACPI. Этот выбор отключает поддержку ACPI в ядре, однако многие особенности влияющие на быстродействие все равно включены, напримерe DMA для IDE жестких дисков.

Установка - локальный APIC выключен: Если обычная установка не получается, возможно ваше оборудование не поддерживает APIC. Если это так, используйте эту опцию для установки без поддержки APIC.

Установка - безопасные параметры настройки: Если у вас не получается с Установка, этот выбор может помочь вам.

Спасательная система: Этот загрузочный образ запускает маденькую Linux систмеу в RAM. Это полезно если ваша система запускает не правильно. После загрузки этой системы, войдите как root.

Проверка программно-аппаратных средств: Запускает инструмент проверки BIOS, который проверяет ACPI и другие части вашего BIOS.

Тест памяти: Тест памяти более полезный, чем просто проверка новых модулей памяти. Это стрессовая проверка большей части вашей компьютерной системы и может выявить проблемы в оборудовании.

Параметры загрузки: Параметры загрузки могут полностью изменять поведение вашей системы. Это настройки ядра.

F1 Справка: Зависит от содержания. Будут показаны разные экраны в зависимости от активного элемента экрана загрузки. Также есть описание этой системы помощи.

F4 Язык: Установить язык и раскладку клавиатуры используемые загрузчиком.

F2 Видео режим: Выберите необходимое разрешение экрана, используемое при установке. Если у вас есть проблемы с графической установкой, текстовый режим может быть поможет вам.

F3 Источник установки: Выберите источник установки.

F5 Обновление драйвера: Для очень новых машин, системе может понадобиться драйвер обновления.

Обновление драйвера

Если вам необходима дискета или CD-ROM с драйвером обновления, нажмите F5. Загрузчик попросит вас вставить носить драйвера обновления после загрузки ядра Linux.

Драйвер обновления это обычно дискета с новыми версиями драйверов оборудования или исправлениями ошибок необходимыми во время установки.

Загрузка с жёсткого диска

Выберите Загрузить установленную ОС чтобы запустить систему установленную на вашем жестком диске. Эта система должна быть установлена, так как стартует только MBR (Главная загрузочная запись) первого жесткого диска. ID устрайства перевого жесткого диска предоставляет BIOS компьютера.

Используйте это, если вы забыли вытащить CD или DVD из дисковода и вы хотите запуститькомпьютер с жесткого диска.

Установка - ACPI выключен

Очень новое оборудование иногда требует ACPI для управления прерываниями. ACPI полностью заменяет старую APM систему.

Выберите Установка - ACPI выключен если есть проблемы с ядром во время загрузки. Известные проблемы с машинами, которые имеют проблемы с AC:

* ядро приостанавливается при загрузке
* PCI платы не обнаружены или неверно инициализированы

Вы можете попробовать использовать Проверка программно-аппаратных средств параметр загрузки и посмотреть результат проверки ACPI.

Не путайте это с ACPI (Усовершенствованный интерфейс конфигурирования системы и управления энергопитанием).

Параметры загрузки

Есть два типа опций загрузки. Во первых, это опции которые влияют на установщик. Во вторых, это опции ядра. Некоторые из наиболее распостраннеых опций:
a) опции установщика

установить -- выберите источник установки
параметры сети -- сетевые параметры
параметры vnc -- параметры для установки через VNC

b) параметры ядра

заставка -- влияет на поведение экрана заставки
apm -- переключает управление питанием
acpi -- усовершенствованный интерфейс конфигурирования системы и управления энергопитанием
ide -- управляет IDE подсистемой

Параметры установки: install

По умолчанию, локальный CD-ROMы являются источником установки. Для сетевой установки, выберите параметр установить. Возможные сетевые протоколы
* FTP
* NFS
* HTTP
Используемый синтакс это просто стандартынй URL. Дл примера, если ваш сервер находится по адресу 192.168.0.1 и вы хотите осуществить установку с NFS из каталога /install на этом сервере, укажите источник в следующем виде:

install=nfs://192.168.0.1/install

Сетевая карта будет настроена с dhcp или вы сами должны указать параметры как описано в параметры сети.

Параметры ядра: splash

Экран заставки это картинка показываемая вовремя запуска системы.

splash=0

Экран заставки выключен. Это полезно со старыми мониторами или когда происходят ошибки.

splash=verbose

Активируется экран заставки, сообщения ядра и загрузки по прежнему показываются.

splash=silent

Активирует заставку, но без сообщений. Однако будет показ процесс выполнения.

Параметры установки: Параметры сети

Можно настроить сетевой интерфейс прямо сейчас. Оборудование будет определено позже YaST2. Минимальный набор параметров вашей сетевой карты состоит из IP адреса хоста и сетвой маски. Для примера:

hostip=192.168.0.10 netmask=255.255.255.0

или в короткой форме:

hostip=192.168.0.10/24

Если вы указали сетевая установка и не указали обе эти опции, установщик попытается настроить сетевой интерфейс с dhcp. Если вам необходим шлюз по умолчанию, укажите это с параметром шлюз. Например:

gateway=192.168.0.8

Параметры ядра: apm

APM это одна из двух стратегий управлений питанием исполюзуемых в настоящее время в компьютерах. Она в основном используется в лаптопах для функций таких как остановка диска, но она также может использоваться для отключения компьютера во время отключения питания. APM завист от правильной работы BIOS. Если BIOS не исправен, APM ограничено в применении или запретит работу компьютера. Однако, она может быть отключена со следующим параметром

apm=off -- полностью выключает APM

Только некоторые очень новые компьютеры имеют преимущество перед другими ACPI.

Параметры ядра: acpi

ACPI (Усовершенствованный интерфейс конфигурирования системы и управления энергопитанием) эвляется стандартом которые определяет интерфейс управления питанием и конфигурацией между операционной системой и BIOS. По умолчанию, acpi включен, если определяется, что версия BIOS датирована позднее чем 2000 г. Есть несколько общих параметров для управления поведением ACPI:

pci=noacpi -- не использовать ACPI для управления PCI прерываниями
acpi=oldboot -- только часть ACPI которая относиться к загрузке, остается активной
acpi=off -- полностью выключает ACPI
acpi=force -- включает ACPI даже если ваш BIOS датирован ранее чем 2000

Особенно на новых компьютерах, заменяет старый apm системы.

Параметры ядра: ide

IDE, в отличии от SCSI, обычно используется в большинстве рабочих станциях. Чтобы предотвратить некоторые проблемы с оборудованием которое случается с IDE системами, используйте параметр ядра:

ide=nodma -- выключить DMA для IDE дисков
